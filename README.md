## ⚙️ One-Time Local Setup

### Step 1 — Put raw CSVs in `data/raw/`

Required files:
```
Crop_and_fertilizer_dataset.csv
Crop_recommendation_with_factors.csv
crop_recommendation_with_soil_nutrients.csv
crop_soil_district_wise.csv
yeild_crop_production.csv

yeild_all_agriculture_related data.of_India_csv
Price_Agriculture_commodities_Week.csv
price_Agriculture_price_dataset.csv
price_commodity_price.csv
```

### Step 2 — Run the pipeline in order

```bash
pip install -r requirements.txt

python module1_module2.py               # ~2-5 min (GridSearchCV)
python module3_price.py                 # ~30s
python module3_arima_module4_profit.py  # ~1-2 min (ARIMA)
python module5_shap.py                  # ~1 min
```

### Step 3 — Test locally

```bash
streamlit run app.py
```

## 🚀 Deploy to Streamlit Cloud

1. Push this repo to GitHub (include `models/` and `data/cleaned/`)
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect repo → Set **Main file**: `app.py`
4. Click **Deploy**

> ⚠️ If `.pkl` files are >100 MB, use Git LFS:
> ```bash
> git lfs install
> git lfs track "*.pkl" "*.h5"
> git add .gitattributes
> ```
